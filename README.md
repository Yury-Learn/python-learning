# Курс «Python с ChatGPT для системного администратора»

идеальный стартовый набор: спокойно с нуля, на русском по возможности, плюс сразу нормальная рабочая среда (VS Code + GitHub), и ты уже не боишься Linux. Значит, я дам тебе готовый учебный план на 4 недели с разбивкой по дням. После 4 недель ты:

- понимаешь базовый синтаксис Python;
- умеешь писать и запускать свои `.py`-скрипты локально;
- работаешь в VS Code;
- коммитишь код в GitHub как нормальный инженер;
- пишешь маленькие полезные утилиты типа парсера логов.

Каждый день я буду говорить:
- что ты делаешь;
- что спрашивать у ChatGPT;
- какой код написать, с объяснением переменных/параметров.

Если мы это пройдём, то у тебя будет портфолио в GitHub, а не просто «я читал теорию».

Репозиторий: `https://github.com/Yury-Learn/python-learning`  
В проекте используем **Jupyter** (ноутбуки) вместе с VS Code.

## 1. Для кого этот курс

- Для системных администраторов и начинающих DevOps, которые:
  - уже немного знают Linux (CLI, базовые команды);

## 3. Рабочая среда

### 3.1. Минимальные требования

- ОС:
  - Linux (желательно) или Windows 10/11 + WSL2 с Ubuntu/Astra/другим Linux.
- Python: **3.10+** (проверить `python3 --version`).
- Доступ к терминалу (bash/zsh/PowerShell + WSL).

### 3.2. Что установить

1. **Python + venv**
   - Убедиться, что установлен Python 3:
     - `python3 --version`  
   - Установить модуль виртуальных окружений (если нужно):
     - `python3 -m pip install --user virtualenv` или использовать встроенный `venv`.

2. **VS Code**
   - Установить Visual Studio Code.
   - Обязательные расширения:
     - `Python` (Microsoft);
     - `Jupyter` (Microsoft);
     - `GitHub Pull Requests and Issues` (по желанию, но полезно).

3. **Git и GitHub**
   - Установить `git` (`git --version`).
   - Создать аккаунт на GitHub.
   - Склонировать репозиторий:
     ```bash
     git clone https://github.com/Yury-Learn/python-learning.git
     cd python-learning
     ```

4. **Jupyter**
   - Внутри репозитория создать и активировать виртуальное окружение:
     ```bash
     python3 -m venv .venv
     source .venv/bin/activate      # Linux/WSL
     # .venv\Scripts\activate.bat   # Windows (cmd)
     # .venv\Scripts\Activate.ps1   # Windows (PowerShell)
     ```
   - Установить нужные пакеты:
     ```bash
     pip install jupyter
     ```
   - VS Code автоматически увидит это окружение и будет использовать его для запуска ноутбуков и скриптов.
